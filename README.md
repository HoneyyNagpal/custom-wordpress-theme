# Custom WordPress Theme

A custom WordPress theme built from scratch with a clean, modular structure and real-world features.

## Features
- Custom theme (no page builders)
- Modular structure (`inc`, `template-parts`, `assets`)
- Custom front page template
- Custom Post Types: Services, Testimonials
- Theme Customizer support
- Menu support and responsive layout

## Tech
WordPress, PHP, HTML, CSS, JavaScript

## Usage
Place in `wp-content/themes/`, activate, add Services & Testimonials.
